## 0. 重要 — 事前確認事項

fudebako-agent は、ホストである fudebako の「データはクライアントから出さない」方針に対する**意図的な例外**として動作します。利用者は本ソフトウェアを使用することにより、以下の事項を理解し、同意したものとみなされます。

(a) 利用者がエージェントに送信したプロンプト、ならびにエージェントがツール実行を通じて参照したファイル内容 (`/drive/` 配下のファイル等) を含む情報は、利用者が指定したエンドポイント (既定では OpenRouter (https://openrouter.ai)) 並びに利用者が選択したモデルの上流プロバイダ (Anthropic、OpenAI、Google 等) に対して、HTTPS 経由で送信されます。

(b) 上記送信先サービスの利用規約、プライバシーポリシー、データ保持方針、課金、および**送信されたデータが当該サービス提供者によりモデルの学習その他の目的に利用される可能性**は、利用者と各サービス提供者との間で直接適用されます。著作者は当該サービスの中継、再販、代理、または保証を行いません。

(c) `OPENROUTER_API_KEY` 等の API キーは利用者がブラウザ内ストレージ (`/drive/`) に保存するものであり、配布物には含まれません。鍵の取得、保管、失効、不正利用への対応は利用者の責任です。

(d) 利用者がエージェントとの対話セッションを行うと、プロンプト、AI 応答、ツール入出力を含む会話履歴が利用者のブラウザ内ストレージ (`/drive/.fudebako-agent/history/`) に保存されます。利用者はブラウザを共有する環境やデバイス紛失時に当該履歴が第三者に露出するリスクを了解するものとします。履歴は利用者の操作によりセッション一覧 UI から削除可能です。

(e) 利用者は、第 3.2 条に定める第三者サービス (OpenRouter および利用者が選択した上流モデルプロバイダ) の利用規約を遵守する責任を負います。これには各サービスが定める利用年齢要件、ならびに利用者の所在地および利用先地域に適用される輸出規制法令 (米国 EAR、日本国外国為替及び外国貿易法等) の遵守を含みます。

加えて、利用者は以下の事項について明示的に了解するものとします。

(a) **AI 出力の正確性**: fudebako-agent が表示する応答は、大規模言語モデルによる生成物です。事実誤認、ハルシネーション (もっともらしいが誤った内容)、不適切または有害な出力を含む可能性があり、その内容の正確性、最新性、適法性、特定目的への適合性について、著作者は一切保証しません。重要な意思決定への依拠は利用者の責任において行ってください。

(b) **ツール実行の副作用およびプロンプトインジェクション**: fudebako-agent はモデルの判断に基づき、Python コード実行 (`run_python`)、ブラウザ内ストレージへの読み書き (`read_drive` / `write_drive` / `list_drive`) 等のツールを呼び出します。これらの実行は Pyodide サンドボックスおよびブラウザのストレージ境界内で行われますが、以下の点に特に留意してください。

- **ネットワーク通信**: Pyodide 内で実行される Python コードは、ブラウザの `fetch` API 等を通じて任意の HTTP/HTTPS 通信を発信することが可能です。ここでいう「サンドボックス」はコード実行環境の隔離を意味し、発信ネットワーク通信を阻止するものではありません。
- **ストレージ副作用**: 利用者の `/drive/` 配下のデータの書き換え・削除を含む副作用が発生する可能性があります。
- **プロンプトインジェクション**: エージェントが `read_drive` 等で読み取ったファイル内容、または取得した外部リソースの内容に、モデルへの指示として解釈されうる文字列が含まれていた場合、利用者の意図しないツール呼び出し (任意 Python コード実行、`/drive/` 配下のデータの送信を含む) が誘発される可能性があります。

利用者はツール実行の結果および副作用を自己の責任において確認するものとします。著作者は、信頼できないソースに由来するファイルや外部リソースをエージェントに参照させる場合、特段の注意を払うこと、および取り扱いに注意を要するデータ (秘密鍵、認証情報、個人情報等) を `/drive/` 配下に置かないことを推奨します。

(c) **スケジュール実行**: 本ソフトウェアのスケジュール機能は、利用者の指定時刻に同一ブラウザ UI 上でエージェントの再実行を行うものです。実行はブラウザタブが開いている間に限定されますが、利用者の意図しないタイミングでツール呼び出しおよび外部 API 課金が発生する可能性があります。
